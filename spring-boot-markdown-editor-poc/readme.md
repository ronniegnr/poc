## spring-boot-markdown-editor-poc

### Project Info

- Spring Boot
- Gradle
- Hibernate
- Spring Data
- H2
- Editormd as Markdown editor
- Commonmark as Markdown parser

### Project Commands

- Run the project with -
        
    - gradle bootRun
    
- Access the application with **localhost:8080**

- Access H2 web console with  **http://localhost:8080/h2-console/login.jsp**. Make sure **jdbc:h2:mem:testdb** is given as JDBC URL and use **password** as password.
